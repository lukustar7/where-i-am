# Tactical GPS Dashboard

An offline-capable, single-file H5 GPS dashboard designed for mobile browsers.

## Features

- **Compass**: Real-time direction with low-pass noise filtering and Android/iOS sensor support.
- **GPS Metrics**: Tracks coordinates, altitude, accuracy, speed, and heading.
- **Sovereignty Geofencing**: Detects specific regional boundaries to display deviated offset coordinates and dynamic map routing links.
- **Zero Dependencies**: Pure HTML, inline CSS, and SVG assets for offline reliability.
- **Wake Lock**: Keeps the screen active during navigation.

## Usage

Serve via HTTPS. Open in browser and tap "ACTIVATE SENSORS" to start.
